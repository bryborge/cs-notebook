# Stacks

## Summary

A stack is a linear data structure used for storing and managing data in a particular order.  The order
may be "Last in, First out" (LIFO) or "First in, Last out" (FILO).

```
+---+
| n |  <-- Top (Last Pushed)
+---+
|...|
+---+
| 3 |
+---+
| 2 |
+---+
| 1 |  <-- Bottom (First Pushed)
+---+
```
